## 문제

n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

___

## 입력

첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.
___

## 출력

1부터 n까지 합을 출력한다.
___

## 예제 입력 & 출력 1

| 예제 입력 | 예제 출력 |
|-------|-----|
| 3   | 6 |

---

## 해결방안
**1. 1부터 $N$까지 반복하며 변수에 값을 누적하여 더하거나, 등차수열의 합 공식인 $\frac{n(n+1)}{2}$을 사용하여 결과를 도출한다.**
